# Portfolio Website

A modern, professional portfolio website built with cutting-edge technologies.

## Features

- **Responsive Design** - Works perfectly on all devices
- **Dark Mode** - Built-in theme switching
- **Blog System** - Markdown-based blog with syntax highlighting
- **Project Showcase** - Display your best work
- **Contact Form** - Easy way for clients to reach you

## Technologies Used

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Markdown

## Implementation Details

The website uses Next.js 14's App Router for optimal performance and SEO. The design system is built with CSS custom properties for easy theming.
